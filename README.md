# dapodik-webservice

SDK Python Web Service aplikasi Dapodik

## Install

Pastikan python 3.7 terinstall, kemudian jalankan perintah di bawah dalam Command Prompt atau Powershell (di Windows + X):

```bash
pip install --upgrade dapodik
```
